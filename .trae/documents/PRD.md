# 数字乡村桌面游戏 - 产品需求文档

## 1. 产品概述

一款轻量级乡村经营模拟游戏，通过玩家在地图上建设各类设施、完成不同任务、管理资源和村民，来体验乡村治理与产业发展的完整过程。游戏采用桌面游戏风格的视觉设计，融合中国传统乡村元素与现代扁平化美学，让玩家在轻松愉快的氛围中感受乡村振兴的魅力。

### 主要目标用户
- 对乡村振兴、乡村治理感兴趣的用户
- 喜欢经营模拟游戏的玩家
- 希望了解乡村发展流程的教育场景用户

### 核心价值
- 通过游戏化方式普及乡村治理知识
- 展示资金、人力、物资等资源的合理配置
- 体现村民满意度和生态可持续发展的重要性

## 2. 核心功能模块

### 2.1 五大主界面

#### 2.1.1 地图界面（核心）
- **功能描述**：展示乡村全景地图，玩家可在空白地块上建设设施
- **可建设施**：
  - 农田：产出生鲜食材，增加季节性收入
  - 民宿：接待游客，获取旅游收入，提升村民就业
  - 仓储：增加物资存储上限，减少损耗
  - 卫生站：提升村民健康水平，增加满意度
  - 文化广场：丰富村民文化生活，大幅提升满意度
- **交互操作**：
  - 点击空白地块打开建设选择面板
  - 点击已建设施查看详情和升级选项
  - 拖拽缩放地图查看全貌
  - 显示设施状态（正常/维护中/升级中）

#### 2.1.2 任务界面
- **功能描述**：接收和处理各类乡村治理任务
- **任务类型**：
  - 环境整治：清理垃圾、绿化植树、污水处理
  - 助农销售：帮助农户销售农产品、开拓市场
  - 老人关怀：走访慰问、医疗服务、生活帮助
  - 防汛巡查：检查堤坝、储备物资、组织撤离演练
- **任务机制**：
  - 每日自动刷新3-5个新任务
  - 每个任务有完成时限和奖励
  - 任务完成后获得资金、人力或物资奖励
  - 未完成任务会降低村民满意度

#### 2.1.3 资源界面
- **功能描述**：管理系统中的各类资源
- **资源类型**：
  - 资金：用于建设和升级设施，用于支付人力成本
  - 人力：村民劳动力数量，决定任务完成效率
  - 物资：建筑材料、医疗用品等消耗品
  - 季节产出：春播秋收，根据建设的农田数量和季节变化
- **季节系统**：
  - 四季轮换，每季30天
  - 不同季节有不同的任务和产出
  - 冬季产出较少但有取暖需求

#### 2.1.4 村民界面
- **功能描述**：管理和服务村民，了解村民需求
- **村民信息**：
  - 满意度：综合评价，影响任务接受度和收入
  - 技能：农业技术、医疗技术、服务技能等
  - 需求：住房、教育、医疗、娱乐等
  - 参与情况：村民参与公共事务的积极性
- **村民行为**：
  - 村民会提出需求和建议
  - 高满意度村民更积极参与任务
  - 技能高的村民可分配到特定岗位

#### 2.1.5 成果界面
- **功能描述**：展示游戏进度和成就
- **统计指标**：
  - 总收入：累计获得的资金
  - 公共服务覆盖率：卫生站、文化广场等服务村民的比例
  - 生态评分：绿化率、污染治理、环境卫生的综合评分
  - 风险事件：发生的自然灾害、疫情等突发事件次数
  - 年度评价：根据以上指标给出年度考核等级（S/A/B/C/D）
- **存档功能**：
  - 手动保存游戏进度到本地存储
  - 支持读取存档继续游戏
  - 自动保存每季度结束时的状态

### 2.2 游戏流程

```
开始游戏 → 第一季度
  ↓
建设设施 → 接受任务 → 管理资源 → 照顾村民
  ↓                    ↓
继续下一季度 ← 季度结算 ← 年度评价
  ↓
游戏结束（达到10年或满足特定胜利条件）
```

## 3. 界面设计风格

### 3.1 视觉风格
- **整体风格**：扁平化设计 + 手绘插画风格
- **色彩方案**：
  - 主色调：#2D5016（深林绿）代表生态和自然
  - 辅色调：#F5E6D3（米黄色）代表乡村田园
  - 点缀色：#E67E22（橙黄色）代表丰收和活力
  - 背景色：#FDFBF7（暖白色）营造温馨氛围
- **字体选择**：
  - 标题：思源宋体（传统感）
  - 正文：思源黑体（现代感）
- **图标风格**：线性图标 + 少量填充，融入中国乡村元素

### 3.2 布局方式
- **导航**：顶部Tab导航栏，固定5个主界面入口
- **内容区**：左侧固定侧边栏（地图快捷操作），右侧主内容区
- **响应式**：优先桌面端，移动端可折叠侧边栏

### 3.3 交互动效
- 页面切换：平滑淡入淡出
- 建设设施：弹性动画 + 粒子效果
- 任务完成：成功打勾 + 金币飞入动画
- 资源变化：数字滚动 + 颜色提示（增加绿色，减少红色）

## 5. 核心算法

### 5.1 季度结算算法
- 计算本季度各类设施产出
- 扣除维护成本
- 更新村民满意度
- 判定是否有风险事件发生
- 生成下季度任务

### 5.2 年度评价算法
- 综合收入、公共服务、生态、风险四个维度
- 权重：收入30%，公共服务25%，生态25%，风险20%
- 根据综合得分评定等级

## 6. 技术实现建议

### 6.1 前端框架
- React 18 + TypeScript
- Tailwind CSS 用于样式管理
- Zustand 用于状态管理
- React Router 用于页面路由

### 6.2 数据存储
- LocalStorage 用于存档持久化
- 游戏状态在内存中管理

### 6.3 性能优化
- 地图使用Canvas或SVG渲染
- 状态变化使用Immer进行不可变更新
- 大量数据使用虚拟列表优化
