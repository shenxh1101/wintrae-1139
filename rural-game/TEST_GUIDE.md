# 数字乡村游戏测试指南

## 快速测试步骤

### 1. 首页加载测试
1. 打开浏览器访问 http://localhost:5173/
2. 应该看到"数字乡村"加载提示
3. 100ms后自动进入游戏主界面
4. 顶部应显示：
   - 标题"数字乡村"
   - 时间显示"第1年春 第1天"
   - 5个导航按钮：地图、任务、资源、村民、成果
5. **预期结果**：立即看到地图界面，不再是空白页

### 2. 建设设施测试
1. 点击地图上的任意空白地块（虚线边框）
2. 右侧打开建设面板
3. 点击"农田"建设（需要500资金）
4. **预期结果**：
   - 资金从5000变为4500
   - 地图上该位置出现农田图标
   - 显示Lv.1

### 3. 设施详情和升级测试
1. 点击刚才建设的农田
2. 右侧打开详情面板
3. 记录显示数据：
   - 等级：Lv.1
   - 产出：100
   - 维护费：50
4. 点击"升级"按钮
5. **预期结果**：
   - 等级立即变为Lv.2
   - 产出立即变为200
   - 维护费立即变为100
   - 花费显示为500

### 4. 切换页面验证数据一致性
1. 切换到"资源"页
2. 查看"农田产出"部分
3. **预期结果**：
   - 显示"2 个农田"（或1个如果只建了1个）
   - 农田产出=基础产出×数量×季节倍率
   - 预计产出与地图详情完全一致

### 5. 任务系统测试
1. 切换到"任务"页
2. 查看当前任务列表
3. 点击某个任务的"接受任务"按钮
4. **预期结果**：
   - 任务状态变为"进行中"
   - 显示剩余天数（通常10-14天）

### 6. 时间推进和任务倒计时测试
1. 点击顶部的"▶ 开始"按钮
2. **预期结果**：
   - 按钮变为"⏸ 暂停"
   - 时间开始推进
   - 任务剩余天数每秒-1
   - 资金根据产出和花费实时变化

### 7. 任务超时自动失败测试
1. 让时间推进直到任务剩余天数=0
2. **预期结果**：
   - 任务从"进行中"移到"失败记录"
   - 失败任务数+1
   - 村民满意度下降

### 8. 成果统计验证
1. 切换到"成果"页
2. 查看统计面板
3. **预期结果**：
   - "完成任务"数=已完成任务
   - "失败任务"数=失败任务（与任务页一致）

### 9. 存档保存测试
1. 完成一些建设或任务后
2. 点击"保存游戏"按钮
3. **预期结果**：
   - 显示"游戏已保存成功！"
   - 存档列表显示新存档

### 10. 存档读取测试
1. 保存游戏后
2. 刷新浏览器页面
3. **预期结果**：
   - 游戏状态完整恢复
   - 日期、资源、建筑、任务进度都保留
4. 点击"读取存档"可加载之前保存的进度

## 常见问题排查

### 问题1：看到空白页
**解决方案**：
1. 清除浏览器缓存
2. 强制刷新（Ctrl+Shift+R）
3. 检查浏览器控制台是否有错误
4. 尝试使用Chrome或Edge浏览器

### 问题2：数据不更新
**解决方案**：
1. 点击"暂停"按钮停止游戏
2. 刷新页面
3. 检查localStorage是否被禁用

### 问题3：存档无法保存
**解决方案**：
1. 检查浏览器是否允许localStorage
2. 尝试使用隐私模式
3. 检查浏览器控制台错误
